Task Manager App

This is a full-stack task management application built as a learning project.
It allows users to manage tasks efficiently and collaborate by assigning tasks to others.

Features

User login & registration
Task creation and assignment
Dashboard, My Tasks and Completed Tasks
Deadline reminders
Light & Dark theme
Responsive design

Tech Stack

React
Node.js & Express
MongoDB
JWT Authenticatio


How to run the project

Both frontend and backend run together using:
npm install
npm run dev
